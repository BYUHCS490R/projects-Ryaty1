Answers the Questions

  1. The form tag has an "action" attribute. What does it do?

    The "action" attribute specifies the URL to which the form data will be sent when the form is submitted.

  2. The form tag has a "method" attribute. What does it do?

    The "method" attribute specifies the HTTP method to use when sending the form data to the server.

  3. What is the difference between a "POST" and a "GET"?

    A "POST" request sends the form data in the request body, while a "GET" request sends the form data in the query string.
