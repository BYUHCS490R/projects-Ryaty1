Images

  • Photo with different color, and smooth should be encoded with JPG.
  • Images with few colors, simple graphics, logos, or animations are best encoded using GIF.
  • The unique of PNG is that it support the transpancy,like we can use photo as non-rectangular sticker.
  • We can resize the photo, click tool, resize image, set parameter to pixel, and change the width of the photo around 300 - 600 pixels
  • Photo editing software preserves image quality and reduces file size, while CSS only changes the display size in the browser and still downloads the original large file, leading to slow load times and wasted bandwidth.
