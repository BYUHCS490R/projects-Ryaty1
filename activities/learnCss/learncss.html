Beginning CSS

1- Inline CSS applies styles directly to an HTML tag, embedded (internal) CSS goes inside a style tag in the HTML head, and external CSS is in a separate .css file that is linked to the HTML.

2- External CSS makes sites faster, easier to update, and keeps code organized. One file controls styles for many pages, so design changes are simple and consistent

3- A selector is used in CSS to choose which HTML elements will get a certain style or rule applied. It tells the browser what part of the webpage to style.
