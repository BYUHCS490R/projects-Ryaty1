Structural Tag

header, nav, main, footer

header: We use it to contain navigation link, logo, and other introductory content.

nav: We use it to contain the navigation links. It could be anywhere in on the web page. nav doesn't pointing to the other website.

main: We use it to contain the main content of the page.

footer: We use it to contain the footer content, such as copyright information.

HTML Validation

It is when we are using a validator to check the HTML code for errors and ensure it meets the standards set by the W3C. It's helpful to ensure that our code is clean and google doesn't have to spend much time questioning it.
