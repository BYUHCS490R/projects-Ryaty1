Internet

The internet is a global interconnected computers network on media, use to communicate, share data and information.

Web

Web is the system of connected documents on the internet. It is the html pages that are accessed via the internet.

How are the Internet and Web different?

Internet is a network that connect computers to share data and information, it is like a highway, while the web on the other side is like the vehicles, a service that run on the internet, used to access information using the web page, the HTML.

ASCII and Unicode

ASCII is a character encoding system that represents only English letters, numbers, and a few symbols, while Unicode is a much larger system that includes more characters for other languages, as well as emojis and special symbols.

What is UTF-8

UTF-8 stands for Unicode Transformation Format. It can be used for encode and decode the characters. It is an advanced version of the ASCII. It encodes the user input and convert into machine understandable and processes. Then it also decodes the generated output from machine code into human understandable code.

URL

A URL is the web address that directs browsers to websites and digital resources online. It tells the web browser where to find a particular item online by specifying its location and how to get it.

Domain Name

The domain name is like the website's address on the internet that people type in to visit a site, making it much simpler than using IP addresses. Using DNS service, it translate human-friendly domain name (vice versa) into ip address so computer can find, and connect.

HTTP

HTTP is the internet's language for requesting and delivering web pages and resources between browsers and servers.

HTTPS

Similar to HTTP, it is the internet's language for requestingand delivering web page and resources between browser and servers, the different is that HTTPS makes web browsing safer by encrypting communication, protecting sensitive information like passwords and credit card numbers from attackers.

IP Address

IP address is like the digital address of a device on the internet or a network. It allows the device to send and receive infomration.

DNS

DNS (Domain Name System) translates human-friendly (vice versa) domain names into IP addresses so computers can find and connect to each other.

W3C Recommendations

W3C Recommendations are the official standards for how the web should work, that the websites and applications will work well for everyone everywhere
